# twitter-bot

Add an `.env` file with below keys (These keys would be generated from twitter developer account):

```
CONSUMER_KEY=""
CONSUMER_SECRET=""
ACCESS_TOKEN=""
ACCESS_TOKEN_SECRET=""
```

to run the program use build script `./build/build.sh {path_to_.env_file}`

```
./build/build.sh ./.env
```
